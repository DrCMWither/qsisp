## qƨisp 是什么？

**qƨisp** （如果你的字体不支持，就叫它 qU+01A8isp。qsisp 并非这门语言的标准名称！）是一门基于 locale 的多语言 Lisp 方言。

它会根据你的系统语言改变：

* 关键字；
* 括号；
* 字符串；
* 注释；
* 你的心理状态！


## 特性

### 基于 locale 的语法

创造你自己的代码：用你自己熟悉的语言写 qƨisp！你不用学 qƨisp，或者 Lisp 的关键词，甚至不一定用学英语：qƨisp 会适配你。
